# D&D Campaigns

A collection of tabletop RPG campaigns run on [D&D Beyond](https://www.dndbeyond.com).

## Structure

Each subfolder is a self-contained campaign:

```
campaign-name/
    lore.md           # World-building, factions, creatures, mechanics
    plot_structure.md # Act structure, scenes, NPC rosters, and per-act todos
    nullshore_draft.md  # Quick-reference summary of core rules and concepts
    todo.md           # Campaign-wide outstanding work
```

## Conventions

- **Pacing target:** 4 hours per session
- **Platform:** D&D Beyond — all stat blocks follow 5e structure; homebrew entries are flagged inline
- **Lore continuity:** `lore.md` is the source of truth for each campaign; check it before writing new content
- **Instruction files:** `claude.md` and `AGENTS.md` contain directives for AI-assisted writing tools
