This project contains multiple campaigns. Each subfolder in the project root is its own self-contained campaign (e.g. nullshore_expanse/). All campaign-specific files (lore, draft, plot structure, etc.) live inside that campaign's subfolder.
When editing this campaign, maintain a target play time of 4 hours.
When any directive in this file changes, update AGENTS.md to reflect the same change, keeping the two files in sync.
When creating or editing the campaign story, always check the campaign's lore.md (located in the campaign subfolder) for continuity and logic. If there is a gap, call it out so it can be filled in.
This campaign is run on D&D Beyond. Where applicable:
- Stat blocks should follow D&D Beyond formatting conventions (5e stat block structure)
- Encounters should be built with D&D Beyond encounter builder constraints in mind
- Magic items, spells, and conditions should use official 5e names and rules where possible so they can be linked or referenced in D&D Beyond
- When creating NPCs or monsters that don't exist in D&D Beyond natively, note that they will need to be added as homebrew
- Stay within what D&D Beyond can actually support — avoid designing mechanics, features, or systems that the platform cannot represent
- If something would be difficult or impossible to implement in D&D Beyond (e.g. custom environmental rules, non-standard action economy, phased bosses with separate stat blocks), flag it explicitly with a warning so the user can decide how to handle it
